# Autonomous Self-Healing Analog Front-End (ASAFE)

An embedded hardware protection platform featuring adaptive C++ control algorithms designed to continuously monitor analog front-end signals, detect parametric deviations, and execute autonomous self-healing recovery loops.

## Repository Overview
* **/src**: Contains the core C++ adaptive control algorithm classes, interrupt service handlers, and calibration matrices.
* **/sim**: Contains Spice simulation logs, hardware signal validation files, and fault injection test benches.
* **/docs**: Contains system block diagrams, protection state workflows, and seminar technical reports.
